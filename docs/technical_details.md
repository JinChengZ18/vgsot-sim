# Technical Details

This document describes the **physical model**, **device parameters**, **example simulations**, and **benchmark results** of the VGSOT-MTJ simulator.

The simulator models **voltage-gated spin-orbit torque switching** in magnetic tunnel junction devices.



------

## 1. Physics Model

The simulator models magnetization switching of the free layer in a magnetic tunnel junction (MTJ) driven by:

- Spin-Orbit Torque (SOT)
- Voltage-Controlled Magnetic Anisotropy (VCMA)
- Thermal fluctuation field

Magnetization dynamics follow the **Landau–Lifshitz–Gilbert (LLG) equation** with additional torque terms.

### 1.1 Landau–Lifshitz–Gilbert Equation

The magnetization vector $\mathbf{m}$ evolves according to
$$
\frac{d\mathbf{m}}{dt}
=
-\gamma \mathbf{m} \times \mathbf{H}_{eff}
+
\alpha \mathbf{m} \times \frac{d\mathbf{m}}{dt}
+
\tau_{SOT}
+
\tau_{STT}
+
\mathbf{H}_{thermal}
$$
where:

| Symbol                 | Description               |
| ---------------------- | ------------------------- |
| $\gamma$               | gyromagnetic ratio        |
| $\alpha$               | Gilbert damping constant  |
| $\mathbf{H}_{eff}$     | effective magnetic field  |
| $\tau_{SOT}$           | spin-orbit torque         |
| $\tau_{STT}$           | spin-transfer torque      |
| $\mathbf{H}_{thermal}$ | thermal fluctuation field |

### 1.2 Effective Magnetic Field

The effective magnetic field includes several contributions:
$$
\mathbf{H}_{eff}
=
\mathbf{H}_{ani}
+
\mathbf{H}_{demag}
+
\mathbf{H}_{thermal}
$$

#### Anisotropy Field

The perpendicular anisotropy field is given by
$$
H_{ani} =
\frac{2 K_{eff}}{\mu_0 M_s} m_z
$$
where:

| Symbol    | Description                        |
| --------- | ---------------------------------- |
| $K_{eff}$ | effective perpendicular anisotropy |
| $M_s$     | saturation magnetization           |
| $\mu_0$   | vacuum permeability                |

### 1.3 VCMA Effect

Voltage-controlled magnetic anisotropy modifies the anisotropy energy barrier.
$$
K_{eff}(V)
=
K_0 + \xi V
$$
where:

| Parameter | Meaning              |
| --------- | -------------------- |
| $K_0$     | intrinsic anisotropy |
| $\xi$     | VCMA coefficient     |
| $V$       | MTJ voltage          |

Applying voltage across the MTJ temporarily **reduces the energy barrier**, enabling lower-current switching.

### 1.4 Spin-Orbit Torque

The effective field generated by spin-orbit torque is
$$
H_{SOT}
=
\frac{\hbar \theta_{SH} I_{SOT}}
{2 e M_s t_f A}
$$
where:

| Symbol        | Meaning              |
| ------------- | -------------------- |
| $\theta_{SH}$ | spin Hall angle      |
| $t_f$         | free layer thickness |
| $A$           | device area          |
| $I_{SOT}$     | SOT current          |

This torque drives deterministic magnetization switching.

### 1.5 Thermal Noise

Thermal fluctuations are modeled as a stochastic magnetic field:
$$
H_{thermal}
=
\sqrt{
\frac{2 \alpha k_B T}
{\gamma \mu_0 M_s V \Delta t}
}
\cdot G(0,1)
$$
where:

| Symbol     | Meaning                  |
| ---------- | ------------------------ |
| $k_B$      | Boltzmann constant       |
| $T$        | temperature              |
| $\Delta t$ | simulation timestep      |
| $G(0,1)$   | Gaussian random variable |

Thermal noise is responsible for **switching stochasticity** and **error rate**.

The simulator uses a set of physical constants and device parameters defined in the implementation (see `constants.py`). constants

These parameters describe the MTJ geometry, magnetic properties, transport properties, and simulation settings.

| Parameter               | Symbol  | Value                 | Unit | Description          |
| ----------------------- | ------- | --------------------- | ---- | -------------------- |
| Vacuum permeability     | $\mu_0$ | $4\pi\times10^{-7}$   | H/m  | Magnetic constant    |
| Elementary charge       | $e$     | $1.6\times10^{-19}$   | C    | Charge of electron   |
| Reduced Planck constant | $\hbar$ | $1.054\times10^{-34}$ | J·s  | Quantum constant     |

Derived quantity used in the LLG equation:
$$
\gamma = \frac{2\mu_0 \mu_B}{\hbar}
$$
where $\gamma$ is the gyromagnetic ratio.

### 2.2 Device Geometry

The simulated device is a circular MTJ on top of a heavy-metal channel.

| Parameter             | Symbol   | Value | Unit | Description              |
| --------------------- | -------- | ----- | ---- | ------------------------ |

| Barrier thickness     | $t_{ox}$ | 1.4   | nm   | Tunnel barrier thickness |

Derived areas and volumes:

Cross-sectional area of the SOT channel
$$
A_2 = w \cdot d
$$
MTJ area
$$
A_1 = \frac{\pi D^2}{4}
$$
Free-layer volume
$$
V = t_f A_1
$$

### 2.3 Magnetic Material Parameters

| ------------------------ | ------------- | ------------------- | ---- | ---------------------------------- |

| Gilbert damping          | $\alpha$      | 0.05                | –    | Damping constant                   |

| Temperature              | $T$           | 300                 | K    | Operating temperature              |

Effective anisotropy field:
$$
H_{eff} =
\frac{2K_i}{t_f M_s \mu_0}
$$
This determines the intrinsic stability of the perpendicular magnetization.

### 2.4 VCMA Parameters

Voltage-controlled magnetic anisotropy modifies the effective anisotropy.

| Parameter        | Symbol  | Value              | Unit    | Description     |
| ---------------- | ------- | ------------------ | ------- | --------------- |
| VCMA coefficient | $\beta$ | $60\times10^{-15}$ | J/(V·m) | VCMA efficiency |

The anisotropy energy changes with voltage:
$$
K_{eff}(V) = K_0 + \beta V
$$
This effect lowers the switching energy barrier during the voltage pulse.

### 2.5 Electronic Transport Parameters

| Parameter                | Symbol       | Value               | Unit | Description              |
| ------------------------ | ------------ | ------------------- | ---- | ------------------------ |
| Heavy metal resistivity  | $\rho$       | $278\times10^{-8}$  | Ω·m  | Resistivity of β-IrMn    |
| Channel resistance       | $R_W$        | computed            | Ω    | SOT channel resistance   |
| Resistance-area product  | $RA$         | $650\times10^{-12}$ | Ω·m² | MTJ resistance area      |
| TMR ratio                | TMR          | 1                   | –    | Tunnel magnetoresistance |
| Bias suppression voltage | $V_h$        | 0.5                 | V    | TMR reduction voltage    |
| Barrier height           | $\phi_{bar}$ | 0.4                 | V    | Tunnel barrier height    |

Channel resistance is computed as
$$
R_W = \frac{\rho l}{wd}
$$
The MTJ resistance depends on the magnetization orientation through the TMR relation.

### 2.6 Spin-Orbit Torque Parameters

Spin-orbit torque strength depends on material and geometry:

| Parameter            | Symbol        | Value    | Unit | Description                |
| -------------------- | ------------- | -------- | ---- | -------------------------- |
| Spin Hall angle      | $\theta_{SH}$ | 0.25     | –    | Spin conversion efficiency |
| SOT current          | $I_{SOT}$     | variable | A    | Applied current            |
| Free-layer thickness | $t_f$         | 1.1      | nm   | Magnetic thickness         |

Effective SOT field:
$$
H_{SOT}
=
\frac{\hbar \theta_{SH} I_{SOT}}
{2 e M_s t_f A}
$$
This field produces the torque responsible for magnetization switching.

### 2.7 Simulation Parameters

| Parameter           | Symbol     | Value             | Unit | Description               |
| ------------------- | ---------- | ----------------- | ---- | ------------------------- |
| Simulation timestep | $t_{step}$ | $1\times10^{-12}$ | s    | Integration timestep      |
| Temperature         | $T$        | 300               | K    | Thermal noise temperature |

The timestep determines the numerical resolution of the magnetization dynamics.

Thermal noise amplitude depends on
$$
\sqrt{
\frac{2\alpha k_B T}
{\gamma \mu_0 M_s V \Delta t}
}
$$
which enters the stochastic LLG equation.

Applying a voltage across the MTJ reduces the anisotropy barrier.

Observed effects:

## 5. Full LLG Derivation

This section gives a compact derivation of the magnetization dynamics used in the simulator.

### 5.1 Precessional Dynamics

Let the unit magnetization vector be
$$
\mathbf{m} = \frac{\mathbf{M}}{M_s}, \qquad |\mathbf{m}| = 1
$$
Under an effective magnetic field $\mathbf{H}_{eff}$, the torque on the magnetic moment is
$$
\mathbf{T} = -\gamma \mathbf{M} \times \mathbf{H}_{eff}
$$
Dividing by $M_s$ gives the precessional dynamics of the normalized magnetization:
$$
\frac{d\mathbf{m}}{dt}
=
-\gamma \mathbf{m} \times \mathbf{H}_{eff}
$$
This term causes precession of $\mathbf{m}$ around the effective field.

### 5.2 Gilbert Damping

Pure precession does not relax the magnetization toward equilibrium, so the Gilbert damping term is added:
$$
\frac{d\mathbf{m}}{dt}
=
-\gamma \mathbf{m} \times \mathbf{H}_{eff}
+
\alpha \mathbf{m} \times \frac{d\mathbf{m}}{dt}
$$
where $\alpha$ is the Gilbert damping constant.

To obtain an explicit equation, move the damping-related term to the left-hand side and solve for $d\mathbf{m}/dt$. The explicit Landau–Lifshitz–Gilbert form is
$$
\frac{d\mathbf{m}}{dt}
=
-\frac{\gamma}{1+\alpha^2}\,\mathbf{m}\times\mathbf{H}_{eff}
-
\frac{\alpha\gamma}{1+\alpha^2}\,\mathbf{m}\times\left(\mathbf{m}\times\mathbf{H}_{eff}\right)
$$
The first term is the precession term, and the second term drives energy dissipation.

### 5.3 Inclusion of Spin-Orbit Torque

For a spin polarization direction $\boldsymbol{\sigma}$, the damping-like and field-like SOT contributions can be written as
$$
\boldsymbol{\tau}_{DL}
=
-\gamma H_{DL}\,\mathbf{m}\times(\mathbf{m}\times\boldsymbol{\sigma})
$$
Thus the full dynamics become
$$
\frac{d\mathbf{m}}{dt}
=
-\frac{\gamma}{1+\alpha^2}\,\mathbf{m}\times\mathbf{H}_{eff}
-
\frac{\alpha\gamma}{1+\alpha^2}\,\mathbf{m}\times\left(\mathbf{m}\times\mathbf{H}_{eff}\right)
+
\boldsymbol{\tau}_{DL}
+
\boldsymbol{\tau}_{FL}
+
\boldsymbol{\tau}_{STT}
$$
In many VGSOT operating regimes, the dominant current-driven term is the damping-like SOT.

### 5.4 Effective Field Expansion

The effective field is written as
$$
\mathbf{H}_{eff}
=
\mathbf{H}_{ani}
+
\mathbf{H}_{demag}
+
\mathbf{H}_{ex}
+
\mathbf{H}_{thermal}
+
\mathbf{H}_{VCMA}
$$
where:

- $\mathbf{H}_{ani}$ is the perpendicular anisotropy field
- $\mathbf{H}_{demag}$ is the demagnetization field
- $\mathbf{H}_{ex}$ is the exchange or bias-like contribution
- $\mathbf{H}_{thermal}$ is the stochastic thermal field
- $\mathbf{H}_{VCMA}$ is the voltage-induced anisotropy modulation

For perpendicular anisotropy,
$$
H_{ani}
=
\frac{2K_{eff}}{\mu_0 M_s} m_z
$$
and with VCMA,
$$
K_{eff}(V)
=
K_0 + \xi V
$$
Therefore, the voltage dependence enters the LLG equation through $\mathbf{H}_{eff}$.

### 5.5 Spherical Coordinate Interpretation

If the magnetization is parameterized by polar angle $\theta$ and azimuthal angle $\phi$,
$$
m_x = \sin\theta\cos\phi,\qquad
m_y = \sin\theta\sin\phi,\qquad
m_z = \cos\theta
$$
then switching corresponds to a transition between two stable out-of-plane states:
$$
m_z \approx +1
\quad \longrightarrow \quad
m_z \approx -1
$$
or vice versa.

In this representation:

- $\theta$ describes the deviation from the perpendicular axis
- $\phi$ describes in-plane precession
- switching occurs when the trajectory crosses the energy barrier and relaxes into the opposite basin

### 5.6 Energy Picture

The magnetic free-energy density can be written schematically as
$$
E(\theta)
=
K_{eff}\sin^2\theta
+
E_{demag}
+
E_{bias}
$$
For perpendicular devices, the two stable states correspond to the two minima near
$$
\theta \approx 0
\quad \text{and} \quad
\theta \approx \pi
$$
VCMA reduces $K_{eff}$, thereby lowering the barrier between the two minima. SOT then drives the magnetization across the reduced barrier.

## 6. Switching Phase Diagram

A switching phase diagram is a compact way to visualize which combinations of control parameters lead to successful reversal.

### 6.1 General Definition

Let the final switched state be defined by
$$
m_z(t_{final}) < m_{th}
$$
for a chosen threshold $m_{th}$, for example
$$
m_{th} = -0.9
$$
Then the switching phase diagram over two control parameters $p_1$ and $p_2$ is
$$
\mathcal{S}(p_1, p_2)
=
\begin{cases}
1, & \text{successful switching} \\
0, & \text{switching failure}
\end{cases}
$$
Typical choices are:

- $p_1 = I_{SOT}$, $p_2 = V_{MTJ}$
- $p_1 = I_{SOT}$, $p_2 = t_{pulse}$
- $p_1 = t_1$, $p_2 = t_2$ for the two-pulse scheme

### 6.2 Common Regimes

A typical $I_{SOT}$–$V_{MTJ}$ phase diagram contains three regions:

| Region                  | Description                      | Physical Meaning                            |
| ----------------------- | -------------------------------- | ------------------------------------------- |
| No switching            | current and/or voltage too small | barrier remains too high or torque too weak |
| Probabilistic switching | near threshold                   | thermal noise strongly affects outcome      |
| Deterministic switching | sufficient VCMA + sufficient SOT | robust reversal                             |

This can be summarized qualitatively as
$$
\text{Switching boundary:} \qquad I_{SOT}^{crit} = f(V_{MTJ})
$$
with
$$
\frac{d I_{SOT}^{crit}}{d V_{MTJ}} < 0
$$
meaning that stronger voltage assistance lowers the required switching current.

### 6.3 Two-Pulse Phase Diagram

For the optimized two-pulse VGSOT scheme, one often studies a phase diagram in the $t_1$–$t_2$ plane:
$$
\mathcal{S}(t_1, t_2)
$$
Interpretation:

- small $t_1$: barrier-lowering pulse too short
- small $t_2$: switching pulse too short
- excessively large total pulse length: unnecessary energy cost
- an intermediate region gives robust switching with improved efficiency

A useful practical constraint is
$$
t_1 + t_2 = t_{total}
$$
so that the design problem reduces to finding the best pulse partition.

### 6.4 Recommended Plot for This Repository

For this simulator, the most informative phase diagrams are:

| X-axis    | Y-axis                            | Meaning                              |
| --------- | --------------------------------- | ------------------------------------ |
| $I_{SOT}$ | $V_{MTJ}$                         | VCMA-assisted switching threshold    |
| $t_1$     | $t_2$                             | optimized two-pulse switching region |
| $I_{SOT}$ | thermal noise level / temperature | robustness map                       |

Each point can be colored by:

- final state $m_z(t_{final})$
- switching probability
- switching delay
- SER

## 7. SER Scaling Law

Switching Error Rate (SER) quantifies the probability that the device fails to reach the desired state under stochastic fluctuations.

### 7.1 Definition

For $N$ Monte-Carlo trials and $N_f$ failures,
$$
SER = \frac{N_f}{N}
$$
A failure is typically defined as
$$
|m_z(t_{final}) - m_{target}| > \varepsilon
$$
where:

| Symbol        | Meaning             |
| ------------- | ------------------- |
| $m_{target}$  | desired final state |
| $\varepsilon$ | tolerance threshold |

### 7.2 Qualitative Scaling with Current

In thermally activated switching, the error rate typically decreases rapidly as the drive current exceeds threshold. A commonly used qualitative form is
$$
SER(I_{SOT})
\sim
\exp\left(
-\frac{\Delta E_{eff}(I_{SOT})}{k_B T}
\right)
$$
where $\Delta E_{eff}$ is the effective barrier under applied drive.

As $I_{SOT}$ increases, the effective barrier is reduced for the undesired trajectory and the correct switching path becomes more favorable, so SER decreases approximately exponentially.

A simple linearized barrier model is
$$
\Delta E_{eff}(I_{SOT})
=
\Delta E_0
\left(1 - \frac{|I_{SOT}|}{I_c}\right)^\eta
$$
for some critical current $I_c$ and exponent $\eta$.

This yields
$$
SER(I_{SOT})
\sim
\exp\left[
-\frac{\Delta E_0}{k_B T}
\left(1 - \frac{|I_{SOT}|}{I_c}\right)^\eta
\right]
$$
for near-threshold operation.

### 7.3 Scaling with Voltage Assistance

VCMA reduces the anisotropy barrier, which changes the effective switching landscape. A simple phenomenological form is
$$
\Delta E_{eff}(V_{MTJ})
=
\Delta E_0 - \lambda V_{MTJ}
$$
Thus
$$
SER(V_{MTJ})
\sim
\exp\left(
-\frac{\Delta E_0 - \lambda V_{MTJ}}{k_B T}
\right)
$$
In practice, VCMA can reduce the required current for a target SER.

### 7.4 Scaling with Pulse Width

For pulse-driven switching, the error rate also depends strongly on pulse duration. A common empirical trend is
$$
SER(t_p)
\sim
\exp\left(-\frac{t_p}{\tau_{sw}}\right)
$$
where $\tau_{sw}$ is an effective switching timescale.

Longer pulses generally reduce SER, but with diminishing returns and increased energy cost.

For the two-pulse scheme,
$$
SER = SER(t_1, t_2)
$$
and an optimal operating region exists where:

- $t_1$ is long enough to lower the barrier
- $t_2$ is long enough to complete reversal
- total pulse energy remains acceptable

### 7.5 Log-Scale Interpretation

Because SER often spans many orders of magnitude, it is usually analyzed on a logarithmic scale:
$$
\log_{10}(SER)
$$
This is especially useful when comparing:

- current scaling
- temperature scaling
- pulse-width scaling
- VCMA-assisted vs non-assisted switching

### 7.6 Practical Interpretation in This Simulator

For this repository, SER scaling laws are most relevant in:

| Case                      | SER dependence                          |
| ------------------------- | --------------------------------------- |
| `ser_sot_no_vcma_thermal` | $SER(I_{SOT})$                          |
| `ser_optimized_vgsot`     | $SER(t_1)$ with $t_2 = t_{total} - t_1$ |

Typical observations are:

1. increasing $|I_{SOT}|$ reduces SER
2. VCMA assistance shifts the SER curve favorably
3. optimized pulse timing can reduce SER at lower current
4. near-threshold operation shows the strongest stochastic sensitivity
